PolyTrack

Download Now

PolyTrack is a low-poly racing game with loops, jumps and high speeds, where every millisecond counts. The game is heavily inspired by TrackMania, where you race against the clock to improve your time on different tracks.

The game includes a level editor where you can create tracks, which you can then export and share with others.

Game instructions:

  • WASD or the arrow keys to drive.
  • R or enter to restart the track.

Editor instructions:

  • Left mouse button to build.
  • Middle mouse button to rotate the view.
  • Right mouse button to pan the view.
  • Shift + mouse wheel to go up or down.
  • R to rotate the selected part.

Menu music: "Sci-fi Theme" by Maou (CC-BY 4.0) - https://opengameart.org/content/sci-fi-theme-1
