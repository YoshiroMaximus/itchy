Hardware Tycoon

Hardware Tycoon is a free simulation/management game where you run your own hardware company, influence the market, and compete with other companies - either only to stay afloat or revolutionize the industry.

Right now it's in rather early stages of development; the only hardware available to develop is the CPU.

The recommended way to play or in case of graphical issues:

  • Download to your machine from the links below, or if you prefer to play in a browser, use either Google Chrome or Microsoft Edge.
  • Download and install Lato fonts .
  • Your video card and browser should support WebGL (pretty much all modern ones do).

Install instructions

Windows : Unzip the archive and run 'Hardware Tycoon.exe'

Mac : Unzip and run 'hardwaretycoon.app'.

NOTE: The macOS version might crash on startup, so if you wish to download the game a better way would be to use the itch client as described here .

Linux : Unzip and run 'nw' after adding execution permissions like so: 

chmod +x nw

NOTE: Depending on your distribution, you might have to run the game from the command line using 

./nw

in the game's directory.

If you're still having problems, check out the troubleshooting guide or seek help on our Discord server.
